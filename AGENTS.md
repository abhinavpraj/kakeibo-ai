# AGENTS.md

## Project Overview

KakeiboAI is a Streamlit-based personal finance application inspired by the Japanese Kakeibo budgeting methodology.

Primary goals:

* Expense tracking
* Savings goal management
* Spending insights
* Reflection-based budgeting

## Technology Stack

* Python
* Streamlit
* SQLite
* Pandas
* Plotly

## Development Priorities

1. Expense tracking
2. Savings goals
3. Dashboard visualizations
4. Insights engine
5. Reflection system

## Coding Guidelines

* Keep implementation simple.
* Prefer readability over optimization.
* Use SQLite for persistence.
* Avoid introducing unnecessary frameworks.
* Focus on MVP functionality.

## Out of Scope

* Authentication
* Multi-user support
* OCR
* Bank integrations
* Mobile applications

## Project Structure

app.py
database.py
insights.py
requirements.txt

* Avoid placeholder implementations.
* Maintain compatibility with Streamlit.
* Keep dependencies minimal.
