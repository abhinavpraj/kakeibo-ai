# KakeiboAI User Manual

## Introduction

KakeiboAI is a personal finance companion inspired by the Japanese Kakeibo methodology.

The application helps users:

* Track expenses
* Set savings goals
* Categorize spending
* Reflect on purchases
* Receive financial insights

## Getting Started

### Launch the Application

```bash
streamlit run app.py
```

### Set Income

1. Open the dashboard.
2. Enter monthly income.
3. Save the value.

### Set Savings Goal

1. Enter target savings.
2. Save the goal.

### Add Expense

1. Enter amount.
2. Select category:

   * Survival
   * Optional
   * Culture
   * Extra
3. Add description.
4. Save expense.

### Dashboard

The dashboard displays:

* Total spending
* Category breakdown
* Savings progress
* Financial insights

### Reflection Journal

For selected purchases, users may enter reflections about spending decisions.

## Troubleshooting

If the application fails to start:

* Ensure Python is installed.
* Ensure dependencies are installed.
* Verify Streamlit is available.

```bash
pip install -r requirements.txt
```
